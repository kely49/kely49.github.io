Proyectos

Plex Management

Aplicación web diseñada para facilitar la gestión a los administradores de servidores Plex. Ofrece la capacidad de administrar usuarios, agregar múltiples servidores, controlar el número de sesiones de usuario para bloquear automáticamente las excedentes, crear cuentas de demostración y una variedad de funciones adicionales para una gestión eficiente del servidor.

MineClick

Juego web basado en la acción de hacer clics y progresar invirtiendo en recursos para generar clics de forma automática. Ofrece características como recursos inicialmente bloqueados que se desbloquean al alcanzar cierta cantidad de clics, formateo de grandes cantidades de clics para una mejor visualización, compras de recursos y bonificaciones integradas, así como animaciones de bloques que se destruyen a medida que avanza la cantidad de clics, entre otras funciones adicionales.

Plattfox

Minijuego 2D desarrollado en Unity, inspirado en el clásico estilo de plataformas como "Super Mario Bros". Presenta mecánicas de movimiento para el personaje principal, plataformas con movimiento entre puntos específicos, plataformas que se desploman al ser tocadas, enemigos que persiguen al jugador hasta cierta zona antes de regresar a su posición original, enemigos que patrullan entre diferentes puntos, un sistema de puntos de control, objetos interactivos y diversas mecánicas adicionales para ofrecer una experiencia de juego completa.

ContrasteWCAG+

ContrasteWCAG+ es una aplicación diseñada para evaluar y mejorar el contraste de colores en interfaces web y aplicaciones móviles, asegurando que cumplan con los estándares de accesibilidad establecidos por las Directrices de Accesibilidad para el Contenido Web (WCAG). La aplicación proporciona herramientas intuitivas y fáciles de usar para evaluar el contraste entre los colores utilizados en el diseño de una interfaz y ofrece recomendaciones para mejorar la legibilidad y la accesibilidad para todos los usuarios
